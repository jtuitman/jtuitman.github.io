→ Code:

  1. Cartan13, MAGMA code for the computations from "Explicit Chabauty-Kim for the Split Cartan Modular Curve of Level 13" tar git.
  2. Coleman, MAGMA code for Coleman integration and effective Chabauty on general curves (with Jennifer Balakrishnan) tar git examples.
  3. diagfrob, MAGMA code to compute p-adic cohomology and L-factors of diagonal hypersurfaces tar git.
  4. goodmodels, MAGMA implementation of the algorithm from "Point counting on curves using a gonality preserving lift" (with Wouter Castryck) tar git examples.
  5. pcc, MAGMA implementation of the algorithm from "Counting points on curves using a map to P^1" and "Counting points on curves using a map to P^1, II" tar git.
  6. deformation, C implementation (using FLINT) of the algorithm from "Improvements to the deformation method for counting points on smooth projective hypersurfaces" (with Sebastian Pancratz) tar git.
